# BOM 管理工具 - 阿里云 ECS 部署指南

## 前提条件
- 阿里云服务器 ECS（操作系统建议：Ubuntu 22.04 / CentOS 7+）
- 已开放 80 端口（阿里云安全组）
- 服务器已安装 Git
- 公网 IP：8.163.100.79

## 一键部署脚本

```bash
# 1. 安装 Docker
sudo yum install -y yum-utils
sudo yum-config-manager --add-repo https://download.docker.com/linux/centos/docker-ce.repo
sudo yum install -y docker-ce docker-ce-cli containerd.io docker-compose-plugin
sudo systemctl enable --now docker

# 2. 配置国内镜像加速（解决 Docker Hub 拉取超时）
sudo mkdir -p /etc/docker
sudo tee /etc/docker/daemon.json > /dev/null <<'EOF'
{
  "registry-mirrors": [
    "https://mirror.baidubce.com",
    "https://docker.1ms.run",
    "https://docker.1panel.live"
  ]
}
EOF
sudo systemctl daemon-reload
sudo systemctl restart docker

# 3. 拉取项目并启动
cd /opt
git clone https://github.com/guojunjiang-collab/bom_tool.git
cd bom_tool
docker compose up -d --build

# 4. 初始化数据库与管理员账号 (admin / admin123)
docker compose exec backend python -c "
from app.models import Base
from app.database import engine
from app.crud import get_password_hash
from app.models import User
from app.database import SessionLocal

Base.metadata.create_all(bind=engine)
db = SessionLocal()
if not db.query(User).filter(User.username == 'admin').first():
    db.add(User(username='admin', password_hash=get_password_hash('admin123'), role='admin', real_name='管理员'))
    db.commit()
    print('✅ 管理员已创建: admin / admin123')
else:
    print('✅ 数据库已存在，跳过')
db.close()
"

echo "🚀 部署完成，访问 http://8.163.100.79"
```

## 常见问题

### 1. 拉取镜像超时（Docker Hub 被限流）
- 确保已配置 `registry-mirrors`
- 重启 Docker：`sudo systemctl restart docker`

### 2. 外部无法访问服务器
1. **阿里云安全组**：进入 ECS 控制台 → 安全组 → 添加入方向规则
   - 协议：TCP
   - 端口：80
   - 授权对象：0.0.0.0/0
2. **防火墙**（如开启）：
   ```bash
   sudo firewall-cmd --permanent --add-port=80/tcp
   sudo firewall-cmd --reload
   ```

### 3. 数据库迁移失败
请确认 `docker compose exec backend` 能进入容器，并按步骤 4 初始化数据库。

### 4. 查看日志
```bash
docker compose logs -f backend
docker compose logs -f nginx
```

### 5. 停止/重启服务
```bash
docker compose down
docker compose up -d --build
```

## 本地 Docker 维护操作

### 停止并删除容器（保留数据库）

```bash
cd "C:\Users\guoju\Desktop\BOM Tool"
docker compose down
```

### 彻底清理

| 清理级别 | 命令 | 影响 |
|---------|------|------|
| 删容器 + 网络 | `docker compose down` | 容器删除，数据保留 |
| 删容器 + 数据卷 | `docker compose down -v` | **数据库数据丢失** |
| 删构建镜像 | `docker rmi bomtool-backend` | 下次需重新构建 |

```bash
# 完全清理
docker compose down -v
docker rmi bomtool-backend

# 确认无残留
docker ps -a --filter name=bom_
```

### 重新部署

```bash
cd "C:\Users\guoju\Desktop\BOM Tool"
docker compose up -d --build
docker ps --filter name=bom_
```

### 仅重建后端（代码修改后）

```bash
docker compose up -d --build backend
```

> 前端文件通过卷挂载（`./frontend:/usr/share/nginx/html`），修改即时生效，无需重启。
