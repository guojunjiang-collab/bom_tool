1. **PowerShell 编码陷阱**：GBK 文件用 `Get-Content` 读取再写回会损坏中文，改用 Python 二进制模式
2. **text-file-writer 技能强制入口**：所有文本文件写入必须通过 Python 脚本，不得使用内置 write 工具
3. **备份策略**：昨晚直接覆盖 `index.html` 导致今天的部分工作内容丢失（虽然 Git HEAD 保留了今天改的正确部分）
